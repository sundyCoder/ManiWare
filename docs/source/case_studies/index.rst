Case Studies
***********************
Here we report some case studies that show how to implement different applications
using **ManiWare**.


Introduction of case studies
=============================
The following case studies are intended to introduce the features
of **ManiWare** to readers. We suggest readers to run the examples provided by us and then read the
paper to understand the design concept.
